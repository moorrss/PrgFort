## 🔗 Live

**[https://moorrss.github.io/PrgFort](https://moorrss.github.io/PrgFort)**

## Vad är det här?

En dokumentationsapp byggd för att följa med i kursen och snabbt slå upp:

- Alla **Controllers**, **Repositories**, **Entities**, **DTOs** och **Interfaces** från eShop-projektet
- **C# och ASP.NET Core-begrepp** med djupförklaringar, analogier och kodsnuttar (async/await, DTO, Repository Pattern, JWT, EF Core, LINQ m.m.)
- **Lektionsöversikt** — vad introducerades i varje lektion (L1–L13)
- **Postman Collection** — ladda ner och importera direkt i Postman
- **YouTube-integration** — koppla videor från Michael Gustavssons spellista till rätt begrepp
- **AI-assistent** — ställ frågor om projektet och C#
- **GitHub Sync** — hämtar senaste koden från [Jensen-Courses/SYSA25M-CSharp-Forts](https://github.com/Jensen-Courses/SYSA25M-CSharp-Forts) efter varje lektion
